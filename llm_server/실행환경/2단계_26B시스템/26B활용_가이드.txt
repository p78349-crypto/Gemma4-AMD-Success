[ 26B가 빛나는 구체적 시나리오 ]

  1. 영화/드라마 자막 자연번역
     - NLLB: 일단 빠르게 직역 초안 생성
     - 26B : 초안을 받아 자연스러운 구어체로 다듬기
     - 명령 예시:
       "아래 한국어 번역을 영화 대사처럼 자연스럽게
        다듬어줘. 원문: [일본어], 초안: [NLLB결과]"

  2. 전문 문서 번역 (기술/의학/법률)
     - NLLB는 전문 용어에서 오역 발생 가능
     - 26B: 문맥 + 전문 지식으로 정확한 번역
     - 명령 예시:
       "다음 의학 논문 초록을 정확한 한국어로 번역해줘.
        전문 용어는 한국 의학계 표준 용어 사용."

  3. 다국어 교육 콘텐츠 번역
     - 낙후 지역 교육 자료를 현지어로 번역 시
     - 26B: 어린이 수준 언어로 쉽게 풀어서 번역
     - 명령 예시:
       "이 내용을 초등학생도 이해할 수 있게
        쉬운 스와힐리어로 번역해줘."

  4. 번역 품질 자동 교정 파이프라인
     - 26B가 NLLB 번역 결과를 후처리(post-edit)
     - SRT 100줄을 NLLB로 빠르게 번역 → 26B로 교정
     - 시간: NLLB(5초) + 26B 교정(30초) = 35초/100줄

  5. 번역 외 활용
     - EPUB 내용 요약 후 번역
     - 자막 내용으로 영상 주제 분석
     - 복잡한 번역 규칙을 자연어로 지시

[ 8GB 노트북에서 26B 없이 동일 결과 얻는 법 ]

  26B 없이 NLLB-600M으로도 기본 번역은 가능.
  품질이 부족할 경우:
  - NLLB-1.3B로 업그레이드 (RAM +1.5GB)
  - 번역 후 수동 교정
  26B급 품질을 8GB에서 원한다면:
  - 클라우드 API (OpenAI, Claude 등) 연동 필요

[ VRAM 절약 팁 ]

  IQ3_S  (10.4GB) → 최고 품질, 12GB VRAM 필요
  IQ2_M  ( 9.3GB) → 품질 소폭 저하, VRAM 1.1GB 절약
  IQ2_XXS( 9.2GB) → 최소 품질, VRAM 1.2GB 절약
  서버시작_IQ2M.bat 으로 IQ2_M 모드 실행 가능
